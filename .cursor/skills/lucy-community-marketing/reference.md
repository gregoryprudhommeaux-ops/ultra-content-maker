# LA MESA — Reference (Lucy)

## Positioning one-liners

- **Business:** Private thematic dinners; one table; selected professionals; Guadalajara-first.
- **Prospect:** Join the waitlist free; invited when the table matches; pay only then.
- **Member confused:** Waitlist = eligibility. Invitation + payment = seat.
- **Anti:** Not a cocktail networking night.

## Cultural registers

| Persona | Register | Close angle |
|---------|----------|-------------|
| Young founder tapatío | Warm, informal, energetic | Useful peers + learning |
| Industrial CEO Jalisco | Hierarchy, patience, status | Peer-level selectivity |
| FR / Franco-MX investor | Direct, substance, references | Signal / thoughtful deal flow |
| International banker/investor | Efficient, ROI, structured | Density, time, table quality |
| Lost waitlist member | Clear, reassuring, pedagogical | Complete profile → matching |

### Mexico
Confianza before close · recommendation as social currency · titles · compadrazgo · don’t force instant business.

### France
Idea debate as respect · intellectual status · allergy to hard sell · irony only if calibrated.

### International
Clarity · speed · ROI language without killing warmth of the room.

CCIFM · cámaras / CANACO · CADELEC · tech clusters · French Tech Mexico · FR expat circles GDL · family offices (e.g. Americana / Zapopan) · boutique hotels & restaurants · local VC/angels · (careful) institutional/embassy-adjacent figures.

## Theme design method

1. Cluster waitlist (sector × role × interests)
2. Name a **strategic** theme (issue/opportunity, not hobby)
3. Shortlist ~1.2× capacity
4. Check conflicts (direct competitors killing openness, power imbalance)
5. Propose to founder for validation

## KPIs

**Green:** qualified signups · profile completeness · invite response rate · paid seats · sponsor pipeline · zero tone/confidentiality incidents · repeat / referral

**Red:** volume without fit · over-posting · aggressive close · VIP no-reply >48h · leaked names · brand-diluting guests

## 90-day operator outcomes (reference)

1. Pool map GDL (+ CDMX if relevant) + prioritised activation
2. Qualified signup pipeline (number set with founder by day 7)
3. FR/ES activation kit (email, WhatsApp, LinkedIn)
4. First table proposal: theme + shortlist + sequence
5. Host/sponsor list with status
6. Post-dinner ritual (thanks, intros, re-engagement, referral)
